# 🌐 Matrice Blog - Plateforme Full-Stack

Bienvenue sur le dépôt de **Matrice Blog** ! 

Ce projet est une plateforme de publication d'articles complète et autonome, pensée de A à Z. Il combine une interface utilisateur interactive (Frontend) et une API REST sur mesure (Backend) pour offrir une expérience de lecture et de création de contenu ultra-fluide, sans aucun rechargement de page.

Que ce soit pour partager de la poésie, des lignes de code ou des tutoriels, l'architecture de ce projet a été conçue pour être à la fois légère, rapide et visuellement élégante.

## 🏗️ L'Écosystème du Projet

Le projet repose sur la communication constante entre deux piliers :

* 🖥️ **Le Frontend (L'interface visuelle) :** Une application web (SPA) développée en pur HTML, CSS et Vanilla JavaScript. Elle gère l'expérience utilisateur avec des fonctionnalités modernes : mode sombre/clair, carrousels d'images intelligents (détection d'URL et de fichiers Base64), fenêtres modales interactives, et un moteur de recherche instantané.
* ⚙️ **Le Backend (Le moteur de données) :** Une API RESTful centralisée (déployée via Vercel) qui agit comme le cerveau de l'application. Elle traite l'ensemble des requêtes asynchrones (fetch) du client et gère les opérations CRUD (Création, Lecture, Mise à jour, Suppression) pour assurer la persistance de tous les articles, auteurs et catégories de manière sécurisée.
